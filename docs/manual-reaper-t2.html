🎛️ Implementación del Algoritmos NewroSync‑11 en REAPER

Este documento describe el script de procesamiento digital de señales (DSP) en formato **JSFX** nativo de REAPER. El código integra la topología matemática de conmutación limpia directamente en el cálculo muestra a muestra.

📐 Integración de la Ecuación Topológica en el DSP

Para evitar vacíos discretos o distorsiones de conmutación al calcular las trayectorias e inversiones armónicas, el bucle de procesamiento implementa la expresión topológica:

\( T = -i^2 \)

En el código JSFX, esto se traduce en un operador multiplicativo de fase estricto (\( T = 1 \)) aplicado a los búferes de retraso de las frecuencias complemento, garantizando un llenado continuo del espacio de fases a medida que aparecen las muestras de audio.

💻 Código Fuente JSFX de Referencia

Copia el siguiente código y pégalo en un nuevo efecto JSFX dentro de REAPER para ejecutar el procesamiento experimental:

desc: NewroSync-11 Dynamic Matrix Processor
slider1:0<-12,12,1>Ganancia Principal (dB)
slider2:440<200,1000,1>Frecuencia Original (Hz)
slider3:0.626<0,1,0.001>Ratio Relacional T

@init
srate_factor = 2 * $pi / srate;
pos = 0;
// Inicialización de búferes para la memoria recursiva distribuida
buf_size = srate * 2;
buf_L = 0; buf_R = buf_size + 1;

@slider
gain = 10 ^ (slider1 / 20);
f_orig = slider2;
// Aplicación de la topología T = -i^2 para determinar la frecuencia complemento
T_topologia = -(-1); // i^2 = -1, por tanto T = 1 como constante simétrica
f_comp = f_orig * slider3 * T_topologia;

@sample
// 1. Captura de la entrada real x(t)
x_L = spl0; x_R = spl1;

// 2. Almacenamiento en la memoria recursiva del búfer
buf_L[pos] = x_L;
buf_R[pos] = x_R;

// 3. Cálculo del llenado de vacíos espectrales mediante el operador de fase de la ecuación
fase_osc = pos * srate_factor * f_comp;
operador_mod = sin(fase_osc) * T_topologia;

// 4. Re-inyección de las copias procesadas y desfases integrados y(t)
spl0 = (x_L + (buf_L[pos] * operador_mod)) * gain;
spl1 = (x_R + (buf_R[pos] * operador_mod)) * gain;

// 5. Avance de posición dentro del ciclo de fases continuo
pos += 1;
if (pos >= buf_size) (pos = 0;);

🔬 Verificación del Relleno Espectral

Al ejecutar este DSP en REAPER, se puede observar mediante un analizador de espectro que las frecuencias complemento no generan picos aislados de error en las transiciones de fase, sino un flujo de acoplamiento armónico constante, demostrando empíricamente que las variaciones operan como un tejido continuo y simétrico.
